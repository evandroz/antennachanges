 CHSFUniverse.DynamicStateA class that specifies the dynamic state of a given rigid body object in the system in a given coordinate frame. Dynamic State data includes position, velocity, (future: Euler Angels, Quaternions, body angular rates) This class replaces the Position Class in prior versions of HSF.
The two coordinate frames used are ECI and LLA. ECI refers to an unchanging coordinate frame which is relatively fixed with respect to the Solar System. The z axis runs along the Earth's rotational axis pointing North, the x axis points in the direction of the vernal equinox, and the y axis completes the right-handed orthogonal system. Units are (Kilometers, Kilometers, Kilometers). LLA refers to the geodetic latitude, longitude, and altitude above the planetary ellipsoid. Units are(Radians, Radians, Kilometers)
